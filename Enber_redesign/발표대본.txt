[ 사이트 파악 ]
제가 리뉴얼 사이트로 선정한 곳은 기업과 고객에게 맞춤 쿠폰을 제공하는 모바일 상품권 플랫폼 주식회사 엔버 입니다.
주식회사 엔버 사이트를 swot분석을 통해 파악해보았구요. 그 중 제가 이 곳을 선정한 가장 큰 이유 두가지를 말씀드리겠습니다.

[ 사이트 선정이유 ]
메인 페이지 본문 영역에 텍스트가 대부분 이미지로 적용되어 있어서 접근성이 고려되지 않았다는 점과 
메인 페이지 길이가 다소 길어 끝까지 사이트를 탐색하지 못하고 이탈자가 발생할 가능성이 높아보였다는 점입니다.

[ 사이트 리뉴얼 계획 ]
그래서 사이트를 보는 사용자와 듣는 사용자에게 동등한 정보를 전달할 수 있도록 웹 접근성을 고려하고
나스미디어와 에이티애널리틱스라는 기업 사이트 두 곳을 벤치마킹하여 핵심 정보들로만 간결하게 구성하기로 하였습니다.

[ style guide ]
사이트의 한글 폰트는 나눔스퀘어를, 영문 폰트는 몬세라트체를 사용하였고
기존 사이트에서는 신뢰감을 주는 푸른 계열을 사용하였지만 모바일 상품권은 실생활에 밀접하게 연결되어 있는
플랫폼이기 때문에 친근한 느낌을 주는 따뜻한 계열 색상이 더 좋을 듯하여 다음과 같이 변경하였습니다.

[ HTML 재구현 ]

html 시멘틱 태그를 사용하고 구역별 헤딩태그를 사용함으로써 웹 접근성을 고려하였습니다.
그리고 어떠한 사이트인지 설명하는 메타태그를 삽입하여 검색엔진최적화를 하였습니다.
